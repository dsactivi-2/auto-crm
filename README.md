# CRM Automation — job-step.com

**System:** Jobstep IT Solutions v1.15.5
**URL:** https://crm.job-step.com
**Sprache:** Kroatisch

## Projektstruktur

```
crm-automation/
├── src/          # Automation-Scripts
├── config/       # Konfiguration & Credentials
├── tests/        # Tests
└── docs/         # Dokumentation
```

## Setup

1. `cp config/.env.example config/.env`
2. Login-Daten in `.env` eintragen
3. `npm install` oder `pip install -r requirements.txt`
